## Cartesian Laplace Field Expansions

So far, we have leveraged the cylindrical symmetry of round lenses to see how electrostatic and magnetic fields near the optic axis satisfying Laplace's equation can be analytically expanded from axial fields, arriving at very simple paraxial equations.
This captured most of the physics we care about for focusing, though notice we only indirectly investigated Larmor rotation in magnetic fields.

To go further, as well as introduce non-paraxial ray tracing, it's pedagogical to switch to Cartesian 3D coordinates.
For an axisymmetric magnetic or electrostatic field with axial component $F(z)$ (either $B_z$ or $E_z$), the Cartesian expansion satisfying Laplace's equation is

$$
\begin{align*}
F_z(x,y,z) &= \sum_{n=0}^{N}
\frac{(-1)^n}{2^{2n}(n!)^2}
(x^2+y^2)^n F^{(2n)}(z), \\
F_x(x,y,z) &= \sum_{n=1}^{N}
\frac{(-1)^n}{2^{2n-1}(n-1)!n!}
x (x^2+y^2)^{n-1} F^{(2n-1)}(z), \\
F_y(x,y,z) &= \sum_{n=1}^{N}
\frac{(-1)^n}{2^{2n-1}(n-1)!n!}
y (x^2+y^2)^{n-1} F^{(2n-1)}(z).
\end{align*}
$$

Here $F^{(k)}(z)$ denotes the k-th derivative with respect to $z$, and the sum over $N$ captures:
- $N=1$, the paraxial field
- $N \gt 2$, spherical and higher-order aberrations

```mathematica
axialLaplaceExpansion[field_, {x_, y_, z_}, order_ : 2] := 
 With[{r2 = x^2 + y^2},
  {
   Sum[(-1)^n/(2^(2 n - 1) (n - 1)! n!) x r2^(n - 1)
      Derivative[2 n - 1][field][z], {n, 1, order}],
   Sum[(-1)^n/(2^(2 n - 1) (n - 1)! n!) y r2^(n - 1)
      Derivative[2 n - 1][field][z], {n, 1, order}],
   Sum[(-1)^n/(2^(2 n) (n!)^2) r2^n Derivative[2 n][field][z], {n, 0, 
     order}]
   }
  ]
```

## Equations of Motion

Given Cartesian electric and magnetic fields $E(x,y,z)$ and $B(x,y,z)$, the ray equations in the paraxial longitudinal gauge are:

$$
\begin{align*}
x''(z) &= -\frac{E_x}{\Phi}
+ \frac{\eta}{\sqrt{\Phi}}\left\{ B_y - B_z y'(z) \right\}, \\
y''(z) &= -\frac{E_y}{\Phi}
- \frac{\eta}{\sqrt{\Phi}}\left\{ B_x - B_z x'(z) \right\},
\end{align*}
$$
with $\eta = \sqrt{\frac{|e|}{2m}}$.

```mathematica
equationsOfMotion[\[Phi]0_, \[Eta]_, order_][\[Phi]field_, Bfield_] :=
  Block[{
   x, y, z,
   Ex, Ey, Ez,
   Bx, By, Bz,
   \[Phi]
   },
  
  {Ex, Ey, 
    Ez} = -axialLaplaceExpansion[\[Phi]field', {x[z], y[z], z}, order];
  {Bx, By, Bz} = axialLaplaceExpansion[Bfield, {x[z], y[z], z}, order];
  
  \[Phi] = If[PossibleZeroQ[\[Phi]field[z]], \[Phi]0 &, \[Phi]field];
  
  {
   x''[z] + \[Phi]'[z]/(2 \[Phi][z]) x'[z] == -(Ex/(
      2 \[Phi][z])) + \[Eta]/Sqrt[\[Phi]0] (By - Bz y'[z]),
   y''[z] + \[Phi]'[z]/(2 \[Phi][z]) y'[z] == -(Ey/(
      2 \[Phi][z])) - \[Eta]/Sqrt[\[Phi]0] (Bx - Bz x'[z])
   }
  ]
```

This formulation:
- is valid beyond the paraxial approximation,
- naturally includes coupling between position and slope,
- reduces to standard paraxial ODEs when fields are linearized.

## Electrostatic Fields

Let's see how order=1 reduces to the paraxial equations for an electrostatic-only field.
Note: We use a zero-evaluating pure-function: (0 &) to trick the function into accepting a purely-electrostatic field.

```mathematica
equationsOfMotion[\[Phi]0, \[Eta], 1][\[Phi], 0 &] // Expand
```

Indeed, if we take one of these equations we see it's identical to our paraxial equation:

$$
r''(z)
+
\frac{\Phi'(z)}{2\Phi(z)} r'(z)
+
\frac{\Phi''(z)}{4\Phi(z)} r(z)
= 0.
$$

Increasing the order argument, naturally introduces higher-order terms which determine aberrations.

::::{tip} Wolfram language source code
:class: dropdown
:open: false

```mathematica

nonParaxialRayTracing[eqns_, {z0_, zf_}, opts___] := 
 Block[{x, y, z, x0, y0, xp0, yp0}, ParametricNDSolveValue[
   Join[
    eqns,
    {
     x[z0] == x0, x'[z0] == xp0,
     y[z0] == y0, y'[z0] == yp0
     }
    ], {x, y}, {z, z0, zf},
   {x0, y0, xp0, yp0},
   opts
   ]
  ]

physicalConstants = <|
   "e" -> QuantityMagnitude[Quantity["ElementaryCharge"], "SIBase"], 
   "m" -> QuantityMagnitude[Quantity["ElectronMass"], "SIBase"], 
   "c" -> QuantityMagnitude[Quantity["SpeedOfLight"], "SIBase"],
   "\[Eta]" :> 
    Sqrt[Abs[physicalConstants["e"]]/(2 physicalConstants["m"])]
   |>;

electrostaticField["schiske"][a_, k_, \[Phi]0_][
   z_] = \[Phi]0 (1 - k^2/(1 + (z/a)^2));

buildRayTracer["schiske", order_] := Block[{
   \[Phi]0 = 5,
   a = 0.025,
   k = 1/Sqrt[2],
   z0 = -0.5,
   zf = 1,
   \[Eta] = physicalConstants["\[Eta]"],
   \[Phi]field,
   eom
   },
  \[Phi]field = electrostaticField["schiske"][a, k, \[Phi]0];
  eom = equationsOfMotion[\[Phi]0, \[Eta], order][\[Phi]field, 0 &];
  nonParaxialRayTracing[eom, {z0, zf}, MaxSteps -> 100000]
  ]

coneOfRays[numAngularRays_, numRadialRays_ : 3, x0_ : 0, y0_ : 0, 
  minimumRadial_ : 1 10^-3, maximumRadial_ : 1 10^-2] := 
 Block[{\[Theta]list, \[Phi]list, initialConditions},
  
  \[Theta]list = 
   Subdivide[minimumRadial, maximumRadial, numRadialRays] // Most;
  \[Phi]list = Subdivide[0, 2 \[Pi], numAngularRays] // Most;
  Flatten[
   Table[{x0, 
     y0, \[Theta] Cos[\[Phi]], \[Theta] Sin[\[Phi]]}, {\[Theta], \
\[Theta]list}, {\[Phi], \[Phi]list}], 1]
  ]

rayTracer["schiske"] = buildRayTracer["schiske", 3];
nonParaxialRays["schiske"] = 
  ParallelMap[rayTracer["schiske"] @@ # &, coneOfRays[24]];

schiskeRays = ParametricPlot3D[
  Evaluate@
   Map[Prepend[Through[#[z]], z] &, 
    nonParaxialRays["schiske"]], {z, -0.5, 1.0}, PlotRange -> All, 
  BoxRatios -> {2, 1, 1}, PlotStyle -> RGBColor[0, 0.5, 0]
  ]

```

## Magnetic Fields

### Larmor Rotation

Trying the same expansion for a magnetic-only field at paraxial order yields a less familiar result:

```mathematica
equationsOfMotion[\[Phi]0, \[Eta], 1][0 &, B] // Expand
```

This is precisely because the axisymmetric version replaced the coupling effect of the Larmor rotation by an effective radial restoring force proportional to $B_z^2$.

Let's quickly derive where this comes from. Our starting point will be the above equations without the cubic terms:

$$
\begin{align*}
  x'' &= - \frac{\eta}{\sqrt{\Phi_0}}B(z) \, y' - \frac{\eta}{2 \sqrt{\Phi_0}}B'(z) \, y \\
  y'' &= \frac{\eta}{\sqrt{\Phi_0}}B(z) \, x' + \frac{\eta}{2 \sqrt{\Phi_0}}B'(z) \, x \\
\end{align*}
$$

```mathematica
larmorFrequency[z_] := \[Eta] B[z]/(2 Sqrt[\[CapitalPhi]0])

paraxialEqnX = 
  x''[z] == -2 larmorFrequency[z] y'[z] - larmorFrequency'[z] y[z];
paraxialEqnY = 
  y''[z] == 2 larmorFrequency[z] x'[z] + larmorFrequency'[z] x[z];
```

Next, we will combine the equations into a single complex-valued equation (where rotation is more naturally expressed), using a dummy variable:
$$
u(z) = x(z) + i\, y(z).
$$

```mathematica
complexVariable[z_] := x[z] + I y[z]

complexVariablePrime = D[complexVariable[z], z];
complexVariableDoublePrime = D[complexVariable[z], {z, 2}];

complexEquation = 
 complexVariableDoublePrime /. {x''[z] -> paraxialEqnX[[2]], 
    y''[z] -> paraxialEqnY[[2]]} // Simplify
```

This is identical to the following complex-valued ODE:
$$
  u'' = 2 i \Omega u' + i \Omega' u
$$
with $\Omega=\frac{\eta B[z]}{2 \sqrt{\Phi_0}}$ being the Larmor frequency.

Now, we can define the Larmor rotation angle and substitute into the ODE:
:::{math}
:label: eq_larmor_rotation_angle
\theta_z = \int \Omega(z) dz
:::

```mathematica
larmorRotation[z_] := Integrate[larmorFrequency[z], z]
complexSubstitution = w[z] Exp[I larmorRotation[z]]

complexSubstitutionPrime = D[complexSubstitution, z];
complexSubstitutionDoublePrime = D[complexSubstitution, {z, 2}];
rotatedODE = 
 Simplify[
  complexSubstitutionDoublePrime - (2 I larmorFrequency[
       z] complexSubstitutionPrime + 
     I larmorFrequency'[z] complexSubstitution)]
```

$$
  \frac{\exp[i \theta_z] \left( \eta^2 B^2(z) w(z) + 4 \Phi_0 w''(z) \right)}{4 \Phi_0} 
$$

Since this is equal to zero, the term in the parentheses must also be zero -- which is precisely the paraxial ODE for magnetic fields.
Magnetic lenses focus through Larmor rotation; in the rotating frame this appears as a quadratic restoring force proportional to $B_z^2$.

### Glaser Ray Tracing

magneticField["glaser"][a_, B0_][z_] = B0/(1 + (z/a)^2);
